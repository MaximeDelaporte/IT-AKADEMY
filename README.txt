Install-vagrant

#QU'EST CE QUE C'EST
install_vagrant est un installateur de machine virtuelle.
Il vérifie et installe VirtualBox et Vagrant si vous ne les avez déjà installé.
Il crée ensuite une machine virtuelle et l'allume.

#INSTRUCTIONS
Créer un dossier dans lequel vous voulez installer votre vagrant.
Déposer le fichier install_vagrant.sh dans ce dossier.
Ouvrer le terminal
Aller dans le dossier qui contient le fichier install_vagrant.sh.
Lancer la commande bash install_vagrant.sh
Suivre les instructions à l'écran.
Si vous n'avez pas rencontré d'erreur, un fichier Vagrantfile a été créé dans votre dossier.
Une machine virtuelle est actuellement allumée.
Pour l'eteindre, tapez dans le terminal la commande vagrant halt
